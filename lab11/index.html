Lab 11: Advanced Block-level Styling I

Challenge

The purpose of this lab was to experiment with block level styling using CSS.

Problems

We did not encounter any major problems during this lab. We merely had to experiment with and adjust the margins, padding, placement and size of the various elements to get the looks we wanted.

Results

We created a lab page that contained styled block elements with maximum and minimum width settings, specific positions, and a homepage link with a sticky position.
