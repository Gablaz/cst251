Lab 16: Fancy CSS II

Challenge

The purpose of this lab was to experiment with the fancy CSS we learned in the lecture.

Problems

The only trouble we ran into was remembering the proper code for the different CSS effects. We referred back to the lecture whenever we had a problem.We also had to experiment with the positioning and layout of the different elements.

Results

We created web pages styled with background images, transitions, animations, and columns.
